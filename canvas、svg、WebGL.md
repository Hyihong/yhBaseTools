
# canvas | SVG | WebGL 大杂烩

## 前言
canvas、svg这些东西都在开发中应用过，WebGL，em...对我来说就是一个熟悉而陌生的名词。今天要来好好捋一捋这些技术各自发挥的作用和之间的关系。

## 名词解释 &&  理清关系
* canvas: 网页画布，完。
* SVG: 矢量图，XML 格式定义图形，最重要的就是放大不模糊，完。
* WebGL : canvas原本只能画2D图形，但是浏览器若支持WebGL技术，则可以绘制3D图形。
